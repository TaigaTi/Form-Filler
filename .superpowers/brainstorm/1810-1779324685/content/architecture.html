Message flow when Ctrl+Shift+F is pressed:
1. Chrome command → service worker receives "fill" event
2. Service worker → content script: "extract fields"
3. Content script → service worker: [{label, type, id, options}]
4. Service worker runs rules engine + optional Claude call
5. Service worker → content script: {fieldId: value} map
6. Content script sets values + fires input/change events
